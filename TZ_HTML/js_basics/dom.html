Terminologie

  • Root-Node => html-Element
  • Parent => Direkte Vorfahr einer Node
  • Child => Direkte Nachfahre einer Node
  • Sibling => Direkter Nachbar einer Node
  • Descendant => Indirekte und direkte Nachfahren einer Node
  • Ancestor => Indirekte und direkte Vorfahr einer Node

Wichtig: Jedes HTML-Element ist ein Node, aber nicht jede Node ist ein HTML-Element
